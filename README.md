# PLATFORMANCE — готовый сайт

Адаптивный корпоративный SPA-сайт без сборщика и внешних библиотек. Его можно открыть локально, опубликовать на GitHub Pages, Netlify, Vercel или обычном хостинге.

## Состав

- `index.html` — основная страница.
- `privacy.html` — политика конфиденциальности.
- `thank-you.html` — страница после отправки формы.
- `404.html` — страница ошибки.
- `assets/styles.css` — дизайн и адаптивные стили.
- `assets/app.js` — интерактивность, валидация и отправка формы.
- `assets/config.js` — настройки интеграции формы.
- `robots.txt`, `sitemap.xml`, `.nojekyll` — файлы для SEO и GitHub Pages.

## Публикация на GitHub Pages

1. Создайте публичный или приватный репозиторий GitHub.
2. Загрузите в корень репозитория **все файлы из этой папки**, сохраняя папку `assets`.
3. Откройте `Settings → Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/ (root)`, затем сохраните.
6. После публикации GitHub покажет постоянную ссылку на сайт.

## Подключение формы к n8n или CRM

Откройте `assets/config.js` и укажите URL webhook:

```js
window.PLATFORMANCE_CONFIG = {
  webhookUrl: 'https://ваш-n8n-домен/webhook/platformance-lead',
  thankYouPage: 'thank-you.html'
};
```

Форма отправляет JSON со следующими полями: `name`, `phone`, `channel`, `marketplace`, `store_url`, `comment`, UTM-метки, URL страницы и время создания. Встроены маска телефона, валидация, honeypot-защита и сохранение черновика в браузере.

До подключения webhook форма работает в демонстрационном режиме: данные не уходят во внешнюю систему, пользователь переходит на страницу благодарности.

## Что обязательно проверить перед публикацией

- телефон, email, адрес, юридическое лицо и реквизиты;
- право публиковать формулировки «5 лет» и «2000+ карточек»;
- реальные кейсы и разрешение клиентов на раскрытие данных;
- финальную редакцию политики конфиденциальности с юристом;
- основной домен в `canonical`, Open Graph, `robots.txt` и `sitemap.xml`;
- webhook формы и уведомления менеджерам.

## Как заменить домен

Найдите в файлах строку `https://uhjr7sv23.urest.org/` и замените её на новый основной домен. После этого обновите `sitemap.xml` и `robots.txt`.

## Аналитика

Перед закрывающим тегом `</head>` можно добавить счётчик Яндекс Метрики. Для целей используйте события клика по телефону, перехода в презентацию и успешной отправки формы. В `app.js` уже предусмотрена единая точка отправки формы, куда можно добавить вызов цели.
